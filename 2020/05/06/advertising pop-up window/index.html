Solve the integration of the self-use program in the advertising pop-up window in the lower right corner of Windows

win clean clean 2020/05/06

Solve the integration of the self-use program in the advertising pop-up window in the lower right corner of Windows (use new methods to update)

Here is an example of the pop-up window of Eye Protector (this pop-up window is a reminder that I asked to pop up, not an annoying pop-up window)

1. Use automatic blocking software

1.1 Use tinder’s pop-up window to intercept

The current experience is very good. I think it’s not necessary to introduce it. If you advertise to others, you can use it directly if you don’t have special needs.

This tool can be found in Security Tools

1.2 Turn on Focus Assistant

The effect is relatively poor. Although it can intercept some windows, it will not be closed directly, but hang in the background, but it is also a method, at least it can prevent accidental closing.

2. Manually delete

2.1 Delete the pop-up window through the task manager

2.1.1 Through the process

According to the name of the pop-up window, directly close it through the task manager (only used for pop-up windows with easy-to-find names, and it is still very troublesome for me to find too many applications like me)

Then right click to find

Right-click to select the location of the punch-in file, and then delete the advertisement file

2.1.2 Through user management

It is similar to using process management, but it is more convenient to look at the file name. Similarly, find the window name, right-click and select the location of the punch-in file, and then delete the advertisement file.

2.2 Close the pop-up window to start automatically

Find the Startup tab in Task Manager

Or press ctrl+r and enter cmd command and enter msconfig to find the service tab

2.3 Use third-party tools to delete

2.3.1 ViewWizard handle wizard

ViewWizard View the source of the pop-up window, just follow the above operations

In ViewWizard3.0, select the red button in the lower left corner, drag it to the window you want to close, and then delete the advertisement file through the corresponding path
